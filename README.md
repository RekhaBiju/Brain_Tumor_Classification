# Brain_Tumor_Classification
This project focuses on classifying brain tumors from MRI images using deep learning models like ResNet50, Vision Transformer (ViT), and a Hybrid ResNet-ViT model. The system compares these models to identify the most accurate and efficient approach for detecting different types of brain tumors automatically.

The dataset used for this project is available at:
https://www.kaggle.com/datasets/masoudnickparvar/brain-tumor-mri-dataset
